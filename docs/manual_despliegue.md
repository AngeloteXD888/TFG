Pasos específicos para ejecutar tu proyecto con Supabase.

Configuración de Supabase:

Crear un nuevo proyecto en el Dashboard de Supabase.

Ejecutar el script SQL (proporcionado en la carpeta /sql) en el SQL Editor de Supabase para crear las tablas y las políticas de seguridad.

Configuración del Entorno:

Clonar el repositorio: git clone https://github.com/AngeloteXD888/TFG.git.

Crear un archivo .env en la raíz del proyecto con tus credenciales:

Instalación y Ejecución:

Ejecutar npm install para instalar las dependencias.

Ejecutar npm run dev para iniciar el servidor de desarrollo local.

Hosting: El frontend se puede desplegar en Vercel o Netlify conectando directamente este repositorio de GitHub.

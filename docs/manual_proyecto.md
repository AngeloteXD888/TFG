Justificación del cambio tecnológico.

Metodología: Desarrollo ágil centrado en la rapidez de respuesta.

Decisión Técnica: Se ha elegido Supabase sobre un backend tradicional para garantizar la escalabilidad y aprovechar el tiempo de desarrollo en mejorar la interfaz de usuario (UI) y la experiencia de usuario (UX).

Estado Actual: El proyecto cubre los dos trimestres, habiendo finalizado el diseño de la base de datos en el primer trimestre y la integración con Supabase en el segundo.
